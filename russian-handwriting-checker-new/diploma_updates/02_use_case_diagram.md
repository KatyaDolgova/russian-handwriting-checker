  УД-14 ←extend— УД-16  (тултипы — при наведении на ошибку)

  УД-14 ←extend— УД-18  (удаление — по желанию)
  УД-14 ←extend— УД-19  (массовое перемещение — по желанию)
  УД-23 ←extend— УД-24  (снятие с публикации — по желанию)
  УД-26 ←extend— УД-27  (удаление версии — по желанию)
  УД-28 ←extend— УД-29  (просмотр версий чужой — по желанию)
  УД-28 ←extend— УД-30  (копирование — по желанию)

Тип «include» (обязательный подшаг):
  УД-09 —include→ УД-08  (перед запуском нужно выбрать функцию)
  УД-11 —include→ УД-09  (сохранить можно только после проверки)
