# Нагрузочные тесты

## Установка

```bash
pip install locust
```

## Запуск

```bash
locust -f load_tests/locustfile.py --host https://ваш-домен.up.railway.app
```

Открыть в браузере: http://localhost:8089

## Сценарии пользователей

| Класс | Описание | Вес |
|---|---|---|
| `AnonymousUser` | Незарегистрированный, просматривает сайт | 3 |
| `ActiveTeacher` | Авторизованный учитель, отправляет проверки | 2 |
| `BurstChecker` | Стресс-пользователь, долбит endpoint проверки | 1 |

## Рекомендуемые конфигурации

### Базовая нагрузка (повседневная работа)
| Параметр | Значение |
|---|---|
| Number of users | 50 |
| Spawn rate | 5 |
| Duration | 5 min |

### Стресс-тест (пиковая нагрузка)
| Параметр | Значение |
|---|---|
| Number of users | 100 |
| Spawn rate | 10 |
| Duration | 3 min |

### Спайк (внезапный всплеск)
| Параметр | Значение |
|---|---|
| Number of users | 200 |
| Spawn rate | 50 |
| Duration | 1 min |

## Целевые показатели

| Эндпоинт | Медиана | 95-й перцентиль | Ошибки |
|---|---|---|---|
| Главная страница | < 300 мс | < 500 мс | 0% |
| `GET /api/functions/` | < 500 мс | < 1000 мс | 0% |
| `POST /api/check/stream` | < 10 сек | < 30 сек | < 1% |

## Что проверяется

- Время ответа статики (nginx)
- Время ответа публичного API
- Устойчивость при одновременных запросах на проверку (дорогой LLM-вызов)
- Корректность авторизации под нагрузкой
- Отсутствие 5xx ошибок
